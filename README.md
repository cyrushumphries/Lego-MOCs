# Lego-MOCs

MOC stands for **M**y **O**wn **C**reation. These creations are no official Lego sets.

This collection contains the instructions to these MOCs, so anyone can rebuild them.

The amazing software **Studio** by [Bricklink](https://www.bricklink.com/v3/studio/download.page) has been used to modelize the builds and creat the instruction manuals.

These instructions can also be found on my [Rebrickable Profile](https://rebrickable.com/users/cyrusHumphries/mocs/)
